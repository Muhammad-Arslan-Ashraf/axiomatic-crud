# axiomatic-crud
A full-featured CRUD application built with React & Axios.

## 🛠️ Tech Stack
- React + Vite
- Axios
- Context API
- Custom Hooks
- TailwindCSS
- useCallback & useMemo

## ✨ Features
- GET — Fetch posts with pagination
- POST — Add new post
- PUT — Update existing post
- DELETE — Remove post
- Loading & Error states
- Pagination — 10 posts per page
